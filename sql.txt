create table category
--------------------------
create table category (
    id int primary key auto_increment,
    name varchar(32) unique not null, 
    photo varchar(255) default 'photo.jpg', 
    detail text, 
    islive boolean default true,
    create_at datetime default current_timestamp 
)

how to drop table?
drop table category

how to insert new rows into table?
there are two different approach to insert rows into table. 

1st approach syntax 
----------------------------------
insert into <tablename> values (value-1,value-2,value-3,value-4,value-5,value-6,[value-n])

example 
insert into products values (0,'PNCL001','Apsara Dark penceil','really dark pencil','Stationary','apsara',100,90,1000,100.0,1,'2025-11-01',CURRENT_TIMESTAMP)

insert into products values (0,'SCAL002','Stainless steal scale','good quality scale','Stationary','natraj',200,170,100,150.0,1,'2024-08-12',CURRENT_TIMESTAMP)

2nd approach syntax 
----------------------------------
insert into <tablename> (field-1,field-2) values (value-1,value-2)

example 
insert into products (product_code,name,description,category,price,stock,is_active) values ('WTR007','Cello water bag','1 Ltr unbreakable cello waterbag','Stationary',500,50,1)

## Aggregate function

Function that works upon numeric field and returns single value is called Aggregate function.
Note : never mix normal field and Aggregate function in single sql statement otherwise you may get wrong output.  
sum(field)
sum function returns sum of given numeric field.

avg(field)
avg function returns average of given numeric field.

min(field)
min function return minimum value of given numeric field

max(field)
max function return maximum value of given numeric field

count()
count function count of rows in table (how many rows table has)

Joins
-----------------------------------------------------------------------------------------
Join operation basically create temporary join between any numbers of tables and display data as single virtual table.

how to do join operation

to join multiple tables, there must be one common field between tables to be used in join.

what is common field?
common field is field in two table which has same data type and same set of values (domain values)

types of joins

INNER JOIN / Equi join
Inner join fetches only matching rows from all the tables using in join. here matching rows means common field in both table has same value in both table

LEFT JOIN
Left join fetches all rows from left side table but only matching rows from right side table. here matching rows means common field in both table has same value in both table

RIGHT JOIN
right join fetches all rows from right side table but only matching rows from left side table. here matching rows means common field in both table has same value in both table

CROSS JOIN
FETCH DATA FROM BOTH TABLE WITHOUT ANY CONDITION
